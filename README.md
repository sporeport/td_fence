# td_fence
tower defence game with material harvesting elements with rpg elements.

## Gems
This game requires the Gosu gem to play


## Progress
The game is still unfinished

but currently features:

* Rock and Stick harvesting
* Rock throwing
* Tower building
* Bug squashing
* And Health
